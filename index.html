Welcome to OraSASH project page

Oracle is providing Active Session History (ASH) for Enterprise Edition of Oracle database above version 10g with Diagnosting and Tuning pack. OraSASH is a open source project and it allow DBA's to collect data about active sessions (ASH) and save it in additional repository for future access. OraSASH is using dynamic views available in any release of Oracle database since version 9 and it is supporting Oracle Express, Standard and Enterprise Edition.
